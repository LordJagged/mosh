Parameters

strstring
proc(lambda (in out) ...)

Returns

output-string.

Code

(define (call-with-string-io str proc)
     (receive (out get-string) (open-string-output-port)
       (let ([in (open-string-input-port str)])
         (proc in out)<br>
         (get-string))))
